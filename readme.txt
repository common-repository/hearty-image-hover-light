Free WordPress plugin that lets you upload any image and assign it a title and on hover a description and read more link.

== Description ==

Hearty Image Hover Light is a free WordPress plugin that lets you upload any image and assign it a title with a Font Awesome icon. On hover, the image can be further customized with a short description and a read more link.

= Product Details =

[Hearty Image Hover Light](https://www.heartyplugins.com/wordpress-plugins/free-plugins/product/hearty-image-hover-light) is the free version of our Hearty Image Hover plugin, so compared to the full version, it has limited features and settings. This plugin has 1 settings instance, 4 content items, simple widgets and shortcodes, so for multiple settings instances, unlimited content items and flexible widgets, [check out the full version](https://www.heartyplugins.com/wordpress-plugins/premium-plugins/product/hearty-image-hover).

== Installation ==

1. After downloading the plugin, install it using the standard WordPress plugin installation. Access PLUGINS/ADD NEW and upload the plugin you want to install.

1. Once installed, you need to activate the plugin and find the settings for the plugin inside SETTINGS menu.

For more details regarding the installation and initial configuration of the plugin, check out our step by step easy to follow [tutorial](https://www.heartyplugins.com/how-to-install-and-set-up-a-plugin).
